# Terralatro
A mod that adds Terraria items into Balatro
## How it works
- Accessories -> jokers
- Permanent buff consumables -> negative jokers
- Consumables -> tarot cards
- Ore tiers -> seals
- Reforges -> enchantments
- Armor sets -> decks
## Planned features
### Jokers
- Mana Crystal
> 20 mana, mana tracks on the joker
>> Always spawns negative
- Heart Crystal
> +20 chips
>> Always spawns negative
